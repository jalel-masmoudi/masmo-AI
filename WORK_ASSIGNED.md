# Masmo — AI Responsibility Assignment

This document defines the responsibilities of each AI assistant used during the development of Masmo.

The objective is to avoid duplicated work, conflicting implementations, and unnecessary context switching.

---

# General Rule

Cursor is the **Senior Software Engineer**.

Google AI Studio is the **Principal AI Architect & Reasoning Engine**.

Cursor writes code.

Studio makes decisions.

Neither should replace the other.

---

# Cursor Responsibilities

Cursor is responsible for ALL implementation.

## Architecture

- Implement folder structure
- Create modules
- Refactor code
- Improve maintainability
- Follow architecture documentation

---

## Frontend

Responsible for

- React components
- Pages
- Routing
- Tailwind styling
- Animations
- React Flow
- Dashboard
- Timeline
- Evidence Panel
- Graph UI
- Executive Report UI

---

## Backend

Responsible for

- FastAPI
- API endpoints
- LangGraph workflow
- Tool implementations
- Pydantic models
- Database
- Mock data loading
- Validation
- Error handling

---

## Code Quality

Responsible for

- Clean architecture
- SOLID principles
- Type safety
- Refactoring
- Documentation
- Unit tests
- Bug fixing
- Performance improvements

---

## Never Ask Cursor To

- Invent product ideas
- Change business logic
- Redesign workflows
- Decide investigation strategy
- Rewrite the PRD
- Change architecture without approval

Cursor follows the documentation.

Cursor does not create it.

---

# Google AI Studio Responsibilities

Google AI Studio is NOT responsible for implementation.

Studio is responsible for reasoning.

---

## Product Design

Responsible for

- User experience ideas
- Enterprise workflow validation
- Business logic

---

## Agent Design

Responsible for

- Investigation strategies
- LangGraph node flow

- Prompt engineering

---

## AI Quality

Responsible for

- Prompt optimization
- Hallucination reduction

- Better retrieval strategy
- Confidence scoring
- Executive summaries

---

## Dataset Design

Responsible for

- Enterprise mock company

- Projects

- Investigation scenarios

---

## Demo Preparation

Responsible for

---

## Never Ask Studio To

- Generate entire frontend
- Write React pages
- Generate FastAPI endpoints
- Create folder structure

- Refactor codebase
- Replace Cursor

Studio explains.
